SACM

SACM (Sistema de Administración para Centros Médicos) es un proyecto desarrollado con Java, Spring Boot y MySQL, enfocado en la gestión de pacientes, usuarios y datos clínicos en una IPS. Incluye funcionalidades de CRUD, control de accesos y conexión a base de datos, ofreciendo una solución eficiente y segura para entornos de salud.

SOUNDAL

SOUNDAL es una interfaz de escritorio desarrollada con Java en NetBeans, conectada a una base de datos MySQL. Está enfocada en la gestión de contenido multimedia y funcionalidades específicas para el usuario. Incluye operaciones de CRUD, una interfaz amigable y un diseño centrado en la eficiencia y facilidad de uso en entornos locales.
